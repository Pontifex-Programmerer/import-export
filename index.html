Import/Export

the export keyword can be used to export variables of any kind from a javascript module.
the import keyword can be used to import any variable that is exported from another .js file.
To use import within a script, it must be declared asa module within the scripttag, like this:

HTML

<script type = "module" type = module src = "main.js" > </script>

Javascript

At the top of your javascript file, main.js add the following snippet

import math from "./auxiliary_scripts/math.js"
